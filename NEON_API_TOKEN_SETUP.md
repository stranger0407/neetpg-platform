## 🎯 Instructions (Detailed)

### Access Your Neon Account

1. **Go to**: https://neon.tech/app/settings/api-keys
   - Log in if needed
   - Click "Settings" (bottom left)
   - Click "API keys"

2. **Create New API Key**
   - Click **"Create API key"** button (top right)
   - Enter name: `neetpg-cli-token`
   - Click **"Create"**

3. **Copy the Token**
   - You'll see: `plj_xxxxxxxxxxxxxxxxxxxxxxxx`
   - Click **"Copy"** button
   - Token is now in your clipboard

4. **Return Here**
   - Paste the token in your next message
   - I'll use it to connect

- ✅ Safe: Token is project-scoped (only accesses your database)
- ⚠️ Keep private: Don't share token publicly
- ⚠️ Expire it later: Can delete/revoke from settings
- ✅ Revocable: Can create multiple tokens
- ✅ Limited: Token only has database access, not account access
