# NEET PG Pro - MCQ Practice Platform

Production-grade NEET PG MCQ practice platform with 50,000+ questions, adaptive learning, spaced repetition, and AI-powered explanations.

## Tech Stack

- **Frontend**: React.js (Vite), TailwindCSS, React Router, Recharts
- **Backend**: Spring Boot 3, Spring Security, JWT, Spring Data JPA
- **Database**: PostgreSQL
- **Infrastructure**: Docker, GitHub Actions CI/CD

## Quick Start

### Prerequisites
- Java 21
- Node.js 20+
- PostgreSQL 16+

### Backend
```bash
cd backend
mvn spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Docker
```bash
docker-compose up
```
